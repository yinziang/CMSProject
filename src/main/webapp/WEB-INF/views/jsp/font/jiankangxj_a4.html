饮食管理

日期：2016-03-23

透析病人的饮食原则

  •  足够的热量

由于肾友在饮食上有一些限制，常会这个不敢吃那个不敢吃，热量摄入充足可防止组织蛋白质分解，提高蛋白质的利用率。但如果热量摄取不足，身体将逐渐消瘦，进而使得抵抗力变差，甚至产生更多肾毒素。因此，充足的热量摄取对肾友来说是非常重要的。

  • 适量的蛋白质摄取

肾友在透析过程中容易有氨基酸和蛋白质的流失，故因摄取足够的蛋白质，特别是摄取利用率较高的蛋白质，帮助修补身体组织。

  • 注意控制液体（水分）的摄取

若出现无尿与少尿的情形，需严格限制水分摄取。

  • 注意钠、磷、钾的摄取

钠、磷、钾离子皆需藉由肾脏排出，若肾功能下降，容易使身体内蓄积过多，造成身体或生命的危险，如：过多的钠（盐分）会增加水分滞留，容易导致水肿而增加心肌的负担，过多的磷会导致骨骼病变及皮肤瘙痒，过多的钾会造成心律不整，严重会造成心跳停止而致死。

  • 脂肪摄取原则及标准

对于血脂异常的肾脏病患者，K/DOQI准则建议每天饮食来自脂肪的热量介于每日总热量飞瑞25-35%、饱和脂肪需低于7%、多元不饱和脂肪低于10%，单元不饱和脂肪可高达20%，每日胆固醇少于200mg。膳食纤维20-30mg。饮食中每日增加5-10mg水溶性纤维，约可降低低密度胆固醇3-5%

 

 

透析饮食之控制水分摄入的原则

   

血透病人需严格限制液体的摄取：体内水分过多会导致呼吸急促、高血压、充血性心脏衰竭及肺积水，也容易造成透析中血压下降及抽筋以及透析后的疲惫与无力等危害身体健康的症状。

 

 

控制水分摄取的原则

 

  • 每日总液体摄取量等于每日排尿量——500ml.水分为标准

  • 每日总液体量=每天正常饮食所含的水量+药水+点滴+汤+果汁及饮料+开水等

  • 配合低钠（盐）饮食可以改善口渴的情况。

  • 避免喝过多的汤汁、饮料、粥等水分较多的食物

  • 隐藏性液体来源，如：果冻、冰块、冰淇淋、西瓜等也要注意摄取

  • 先将一日可以喝的水用固定容器装好，将这些水分配饮用

  • 感到口渴时用棉棒湿润嘴唇或漱口或加一点柠檬汁在冰块或饮料中

  • 要喝水时应该以温开水为最佳，一次50-30ml为准则或含冰块，但要限量饮用

  • 固定运动，促进排汗

 

 

如何在家中测量体重

  • 每天在固定的地点，固定的时间，用固定的秤测量体重。

  • 两次透析间以不超过2-3公斤为原则

  • 允许体重增加上限参考表：

透析饮食之摄取足够的热量

   

肾脏病患者应摄取足够热量以维持理想体重，使蛋白质能在体内被完全吸收、利用及发挥应有的功能，可避免因热量摄取不足造成身体组织蛋白质及饮食中蛋白质的分解，进而增加含氮废物的产生

一天的热量需摄入每公斤体重30-35公卡
理想体重计算（公斤）=22*身高（M）*身高（身高）

慢性肾脏病患者可以利用下列食物制作点心或正餐，以增加或补充所需热量
低蛋白：绿豆糕、细米粉、粉条、太白粉、地瓜粉、西米露、凉粉皮、粉圆、玉米粉、藕粉、凉糕等
油脂：建议多利用植物性油脂如橄榄油、高油酸葵花油，芥末油，荠菜油，花生油等，以油炒、油煎食物等方式增加热量
任何食物不可过多食用，如有不了解的食物请咨询血透室医生

 

 

透析饮食之控制磷的摄取
    

当肾功能下降到后期时，体内磷酸盐无法随尿液排出而滞留在体内，使血磷含量增高而血钙浓度下降，促使副甲状腺素分泌增加，骨钙游离到血中，产生骨骼病变，骨头酸疼，皮肤瘙痒和血管钙化等问题

控制血磷的方法——避免摄取高磷的食物
奶类：鲜奶、奶粉、优酪乳、起司、冰淇淋
鱼肉类及其他加工品：各式肉干、肉丝、肉松、鱿鱼丝、带骨吃的鱼类等
内脏类：动物的脑、肝、肾等
蔬菜类：草菇、金针菇、香菇等菇类
五谷根茎类：糙米、五谷米粉、胚芽米、小麦胚芽、燕麦片、麦片、全麦吐司、全麦馒头、杂粮馒头等
坚果类：杏仁、花生、花生酱、芝麻、芝麻酱、腰果、核桃、瓜子等
坚果类：红豆、绿豆、蚕豆、毛豆、青豆仁、豆皮
其他食品：蛋黄、鸡蛋豆腐、巧克力、酵母粉、茶叶、咖啡、奶精等

含磷低的食物
肥猪肉、五花肉、牛肉（半肥）、带鱼、鳝鱼、白萝卜、冬瓜、黄瓜、青椒、茄子、韭黄等
排骨等肉类可先煮水后去汤再炖 

如何食物不可过多食用，如有不了解的食物请咨询血透室医生

 

 

透析饮食之控制钠（盐分）的摄取

 

过量的盐分摄取会容易口渴、水分滞留、体重增加、高血压、水肿、心肺衰竭等，而加速肾脏功能的恶化，所以少盐分摄取是重要降低盐分摄取的技巧
 

 

 

饮食原则

 

1、以自然新鲜的食物为原则，每天摄入量少于5g

  • 减少腌制、烟熏、罐头等加工食品及速食食品的摄取

  • 烹饪时可搭配少量的柠檬、番茄、洋葱或葱、姜、蒜、酒白糖、白醋、肉桂、五香、胡椒等具有特殊风味的食材调味，减少对盐、味精、酱油等钠含量较高之调味料的使用

  • 低钠盐或美味盐中的钾含量高，不建议慢性肾脏病患者使用

  • 零食类、运动类饮料及碳酸饮料也需忌口、如海苔、洋芋片、米果、速食汤等

  • 少用酱料，如番茄酱、豆瓣酱、沙茶酱、辣椒酱等

调味品与钠（盐分）含量换算表
  

1毫克盐

=400毫克钠

=3毫克（1茶匙）味精

=5毫升（1茶匙）乌醋

=6毫升（1.2茶匙）酱油

=12毫升（2.5茶匙）番茄酱

 

2、含钠量高的食物。依类别可分为

  • 奶类：起司

  • 蔬菜类：加工品及腌制品，如：紫菜、海带、榨菜、梅干菜、泡菜、酱菜等

  • 水果类：蜜饯、加工果汁及罐头果汁，如：柿饼、黑枣、芒果干、腌制桃、番茄汁、樱桃汁等

  • 五谷根茎类：麦片、油面、饼干、咸面包等

  • 鱼肉类及其他加工品：腌制的肉，鱼类食品，如：火腿、香肠、腊肉、咸鱼、各种肉类罐头食品、皮蛋、咸蛋、火锅料、贡丸等

  • 其他：速食汤、泡面、海苔、洋芋片、鸡精等

  • 调味料：盐、酱油、味精、味增、豆瓣酱、番茄酱、沙茶酱、蚝油等

  • 多尝试葱、蒜、醋、五香粉等调味料，可增加菜的口感

  • 避免应用浓茶及咖啡

  • 任何食物不可过多食用，如有不了解的食物请咨询血透室医生

 

透析饮食之控制饮食中钾离子的摄取

   

血清中的钾离子过高，会造成病患发生心律不整。避免钾离子摄取过多的方法有：
 A:蔬菜可用清水煮3-5分钟，捞起后沥干有降钾含量的效果，再进行油炒或油拌
 B：含钾量高的蔬菜，如根茎类可以先去皮、切碎，再泡水来降钾含量
 C：禁忌吃生食（包括生鱼片、蔬菜等）未煮熟的食物
 D:汤类含大量的钾离子、不宜多食：蔬菜汤、火锅浓汤、各种各类肉汁及肉汤浓缩的动物来源萃取液（如鸡精、牛肉精等）、咖啡、茶、药草、水果干、果菜汁
 E：避免摄取含钾量高的蔬菜有：香菇、紫菜、榨菜、竹笋、芹菜、红萝卜、菠菜、空心菜、苦瓜、花菜等 

 F：避免摄取含钾量高的水果有：桔子、哈密瓜、枇杷、龙眼、桃子、大枣、芒果、香蕉、奇异果、榴莲、番茄；含钾量较低的水果有苹果、菠萝、柠檬等可吃，单仍需注意水分的摄取
 G：不喝水果罐头内的糖浆。杨桃—禁止食用
 H：任何食物不可过多食用，如有不了解的食物请咨询血透室医生

 


透析饮食之蛋白质的摄入
   

在透析中废物的处理能力大大增加，相对的部分蛋白质也会被透出，因此使用优质蛋白质非常重要的。每日蛋白质摄入达到每公斤体重1.2-1.5g。一个体重60的人，一天可吃90g左右，也就是说只要限制水分及遵守某些注意事项，就能享受近于普通人饮食：
4-6两瘦肉
1两肉=1个鸡蛋=一杯牛奶（240ml鲜奶或全脂奶粉4平汤匙）。若胆固醇高的患者，可以用2个蛋白代替一个全蛋
3碗米饭
1碗饭=2碗面=4片面包=2碗稀饭
由于蛋白质在人体内消化之后，它产生的废物是尿素氮，当肾脏功能失效时，血中的尿素成分会升高，当尿素升高时会出现恶心、呕吐、头痛、腹泻等不适症状，所以我们需要选择在消化以后产生尿素最少的食物，我们称之为优质蛋白。
优质的蛋白质：豆类、鸡蛋、牛奶、瘦肉、河鱼、河虾类
