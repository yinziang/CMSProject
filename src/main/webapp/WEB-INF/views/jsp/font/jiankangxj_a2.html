日常护理

日期：2016-03-25

平日皮肤的护理

 

  • 洗澡时不要将肥皂直接涂在皮肤上，洗完澡后可擦拭乳液

  • 剪短指甲可避免抓破皮肤，预防感染

  • 局部瘙痒时可用冰敷或涂些清凉油

  • 若皮肤瘙痒难耐时，请医师开口服药或外用药

  • 瘙痒厉害时，注意检查血液钙磷及甲状腺激素水平以维持正常

    •  

    透析病人如何纾解情绪压力

     

    • 当意志消沉，心情沮丧时，可将内心的感受表达出来，以减轻内心的情绪压力

    • 可经由社会工作人员获得精神及经济上的协助

    • 与复健良好的个案认识，吸收其经验并可增加信心

    • 肾衰竭患者，通常会有性机能障碍，若因此造成生活不协调及怀孕问题时，可提出与医护人员讨论

    • 参加联谊活动-----有聚会郊游或短期国外旅游，并有医护人员随行。多参加旅游开阔心胸、多吸收一些信息并扩展自己的生活圈

     

    透析病人的运动原则

     

    • 任何透析患者需要经医师评估后，选择适合自己的运动方式

    • 任何运动需要量力而为，决不可勉强为之

    • 糖尿病患者常常有体位性低血压及自主精神病变所引起的心律不齐，运动时要十分小心，另外需要注意血糖变化，运动前先进食，随时携带糖果，以防低血糖的发生

    • 促使红细胞生成素，血红蛋白增加可以提高运动忍耐力，但同时也要注意血清中钾离子会升高，尿素清除率下降，因此仍然要小心与医师配合

     

    透析病人的运动计划

     

    • 运动的种类：多做等张运动，例如：游泳，骑自行车、慢跑，走路对心脏较有帮助。练气功、太极拳亦是值得鼓励的运动。要避免会使血压升高的运动如举重、短跑、跳绳等

    • 运动的长短：每次至少20-30分钟才能达到运动效果，平常未运动者，可依据其耐力逐渐增加运动时间以达到预定的目标

    • 运动的频率：每周至少三次，每次至少40分钟，而且心跳速率须达到最大允许心跳速率的80%（最大允许速率随年龄而异，可请教医师）


    透析病人日常生活的护理重点

     

    • 定时接受透析治疗

    • 适当运动，充足的休息和睡眠

    • 预防感染、养成良好卫生习惯

    • 按时量血压并记录

    • 养成按时排便的习惯，不要便秘

    • 必要时可使用软便剂

    • 注意饮食和水分的摄取

    • 自我观察有无内外出血的情况、自身有无浮肿或加重、大小便颜色、皮肤上有无出血点、瘀斑等

    • 保持身心愉快

    • 保暖；机体受凉收，血管收缩，血压升高。故此，冬天外出易戴口罩；夏天不要赤脚，不要长时间吹电风扇，不要长时间泡在冷水中

    • 居家生活中有任何紧急情况和不适，应立即到门诊、急诊或血液净化中心、病房就诊

     

    透析老年患者的居家安全


       

    老人家在家里最常跌倒的地方依序是客厅，卧室、浴室或厕所，跌倒时的活动以室内走动最多，如果厨房，浴室、楼梯潮湿或地面不平，光线昏暗或到陌生的新环境，穿了不合脚的鞋子，鞋底太滑，也易使老人容易跌倒
        美国运动医学建议老人，每周走路三天，每天累计30分钟，增强肌肉力量，有助于降低跌倒的风险
    居家注意事项

    • 贫血或体位性低血压者，下床时先坐在床边休息3-5分钟，不晕眩才可下床

    • 家中照明及采光应充足

    • 移除家中危险物品，采用无障碍空间设计

    • 在厨房、浴室和浴缸等地方要铺上止滑垫，保持地面干燥。座椅、马桶、床等，不宜易太高太低，软硬需适中

    • 衣物避免太长或太宽，鞋子要合脚及防滑

    • 给与适当的辅助器，如拐杖，助步器及轮椅等

    • 老年人自行如厕或沐浴，须随时注意有无回应

     

    透析病人如何与家人相处

     

    • 透析治疗是一种终身性的治疗方法，这不仅使透析病人个人，也使病人所在的家庭的生活发生巨大的变化，比如家庭开支增加，经常要去医院影响家人上班等等

    • 大多数病人会因此对家人产生一种内疚之情，这种内疚之情是关爱家人，心地善良的表现，是一种积极向上的心态，病人要将这种良好的心态转化为实际行动，以减少对家庭的影响

    • 比如通过自觉限制水分的入量，科学饮食等方法，可以不用增加透析次数，从而减少透析费用，锻炼身体，增强体质，增强抵抗力，从而减少感冒，肺部感染的发生，病情许可的话，多做家务，减轻家人的负担，若有可能，继续坚持工作，自己能够的事不要劳烦家人等等。最重要的是表达出来，一如既往的关心关爱家人

    • 最忌“托病为大”，一味索取，认为家人的付出是应该的，而不情愿在感情上，时间上，体力上再为家人付出

     

    透析病人心理的“8要5不”原则

     

    • 八要

       1.要找个倾吐心事的人：说出心中的焦虑恐惧，并澄清一些错误的想法
       2.要接受患病的事实：身体病了但是心不能贝身体疾病打倒，要有信心，凡自助者天助也
       3.要少抱怨常感恩：因为无法改变的事实会让心情更糟，如果看到自己仍然拥有就会更珍惜生命
       4.要打开心胸抛开疾病苦恼，保持与社会互动，寻求生活的意义，协助家人及关心情况比你更糟糕的人
       5.要时时刻刻怀着自省的心理关怀身边的人;回到自己的岗位，一己之力，作为有用的人挥别心中的阴霾，过着光明的人生不受外界异样眼光的影响，选择快乐和不快乐在你的在于你的一念之差
       6.要坚强的活下去——明天会更好，活着就有希望，为了爱你的人以及你爱的人，站起来积极的活下去，存在本身就是一种价值，不要以为失去了一些东西，人生就没有意义，其实人身没有十全十美的事情
       7.要有宗教信仰，可以是精神有所寄托——弥补心灵的空虚
       8.要培养一些充实生活，服务他人排解无聊

    • 五不

       1.心存不切实际的想法：透析只能帮助排除部分毒素、水分，并无恢复肾功能作用，任何减少透析次数或不透析的想法都不切实际
       2.不要让透析室成为心理压力的地方：透析室是肾友生活的一部分，肾友和工作人员应共同建立一个温馨和谐的环境
       3.不要因为透析与亲友疏远：需要他们的关心与关怀
       4.不要把痛苦的事情藏在心理：找愿意倾听的人宣泄出心中的苦闷
       5.不要过度依赖他人：可以自己做的就不求人，可以减少无助的痛苦，如果不是力所能及的事情也不要勉强自己独立，毕竟没有人能够独立过一辈子。
